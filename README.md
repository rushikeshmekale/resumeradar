# ResumeRadar — AI Resume ATS Analyzer

**Beat the bots. Land more interviews.**

ResumeRadar uses AI to score your resume against any job description the same way Applicant Tracking Systems (ATS) like Workday, Greenhouse, and Lever do — before a recruiter ever sees it.

---

## Features

- **ATS Score** — Instant overall score out of 100
- **Keyword Analysis** — See exactly which keywords you matched and which are missing
- **4-dimension breakdown** — Keyword match, skills coverage, impact & action verbs, ATS formatting
- **AI Suggestions** — Specific, actionable improvements ranked by severity
- **Analysis History** — All your past analyses saved to your account
- **Job Discovery** — Browse matching roles directly
- **PWA Ready** — Install on your phone or desktop like a native app
- **Free** — No credit card required

---

## Stack

- **Frontend** — React 18 + Vite + Tailwind CSS v4 + shadcn/ui → Vercel
- **Backend** — Node.js + Express + MongoDB Atlas → Render
- **AI** — Groq (Llama 3.3 70B)
- **Auth** — JWT email + password
